[TAGS]

Utilite du tag : 
Rechercher toutes les portions de code contenant un tag dans la barre de recherche de VsCode
Naviguer toutes les portions de code contenant un tag sur une page avec ctrl+H

Notation d'un element tag : 
[TAGS] <tagName> <tagName2> ...

Liste des tags du projet :

-	<JWT_tokens_headers> en tetes associes aux JWT Tokens
-	<JWT_tokens_settings> parametrages des JWT Tokens
-	<JWT_tokens_use> utlisation d' elements lies aux jwt tokens

Decrire la nature des tests que l' on souhaite realiser

Utilite du flag [IMPROVE] : les zones du code qui doivent etre ameliorees
